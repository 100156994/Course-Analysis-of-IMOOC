# Course-Analysis-of-IMOOC
基于python,对IMOOC的课程进行数据获取以及分析


## 使用
- 使用bs4，request，re进行IMOOC数据爬取
- pymssql进行数据存储，基于sql server，
- matplotlib和numpy进行数据分析及可视化
